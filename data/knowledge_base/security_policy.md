# Vendor Security Review

Vendors that store, process, or transmit customer, employee, student, financial, or confidential business data require a security review before approval. Acceptable evidence may include SOC 2 Type II, ISO 27001 certification, a completed security questionnaire, and a data processing agreement.

# High Risk Vendors

High risk vendors require review by Security, Legal, and Procurement before contract execution. A purchase order must not be issued until the review team records an accepted risk decision.

# Data Handling

Sensitive data may only be shared with vendors that have a documented data protection agreement, approved access controls, and a named business owner. Data minimization should be applied so the vendor receives only the information needed to perform the service.

# AI Tools

AI-enabled vendors must describe how customer data is used, whether it trains models, where data is stored, and how data can be deleted. If model training on company data is enabled by default, Legal and Security must approve an opt-out or contract restriction.
