GitHub Repository:
https://github.com/0Fin0/procurewise-AI-agent

Project Summary:
ProcureWise is a Track A AI Agent project for ISYS 573. It helps review procurement and vendor-risk requests by combining an LLM, Retrieval Augmented Generation over policy documents, tool-based vendor lookup, approval routing, risk scoring, safety checks, case writing, approval packet generation, evaluation, and human review logging.

Important Note:
The .env file and API key are not included in the public repository. The app can run without an API key in deterministic mode, and it can use live LLM drafting when PROCUREWISE_USE_LLM=true and an OpenAI API key is provided locally.
